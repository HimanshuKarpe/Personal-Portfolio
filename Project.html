Smart Attendance System

This project involves creating a fingerprint-based attendance system using Arduino, ESP32, I2C LCD Display and a fingerprint sensor. The system captures fingerprints, stores them, and allows for attendance marking by recognizing the fingerprints.

Technologies used:

  1. Arduino UNO
  2. ESP32
  3. R307 Fingerprint Sensor
  4. I2C LCD Display
  5. Buzzer
  6. Jumper Cables
  7. HTML/CSS for web interface
  8. PHP

This project was developed as part of my coursework in the 4th semester of my engineering program as a group project.
